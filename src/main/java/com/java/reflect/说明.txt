1.什么是类加载器？它的作用是？
    用来加载通过javac编译成的.class的字节文件，JVM虚拟机中的类加载器负责加载这些类字节文件，
    先经过一系列处理，然后缓存到我们的JVM虚拟机中，提供给程序使用。

2.反射的基础 --> Class类？如何理解Class类
    class：定义一个类的关键字
    Class：描述Java类的组成成分

3.Class类里面含有的内容
    有描述Java类的名字、属性、来自哪个包、类的父类、构造方法、普通方法

4.Class实例化的方法
    我们通常讲的实例化就是讲这个，加载到虚拟机的这个模板字节，创造出一个实例对象出来。
    大写的Class是描述我们的Java类。
    我们怎么实例化我们的Class
    引申出 我们怎么实例化我们的Class
    1） 我们可以拿到实力
    2）我们可以拿到构造方法
5.反射的概念总结
    反射就是把Java类中的组成部分映射成响应的Java对象 字节模板
